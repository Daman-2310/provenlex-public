**Setup before recording:**
- Open these tabs in order: `/` → `/watchlist` → `/obituary/wirecard` → `/book` → `/network` → `/`
- Reduce browser to 16:9 ratio (1920x1080) so the recording looks cinematic
- Use a quiet room, headset mic ideally (not laptop mic)
- Practice the script once before recording — aim for 88-92 seconds total

## Recording tips

- **Speak slower than feels natural.** First takes always sound rushed. Target one word per beat.
- **Pause briefly between sections** — gives the editor cleaner cut points if you want to trim.
- **Don't apologise** if you stumble. Keep going. Energy + confidence > polish.
- **First take is usually best.** Re-record only if you fumble a name or fact.
- **Loom auto-uploads when you stop.** Get the share link, paste into the cold email.
